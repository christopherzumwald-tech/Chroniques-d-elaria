# BLOC 4.1 — Statistiques et Système de jet

## Statistiques (10)

| Abréviation | Statistique | Rôle principal |
|-------------|-------------|----------------|
| FOR | Force      | Puissance physique, blocage, armes lourdes |
| AGI | Agilité    | Esquive, initiative, déplacement |
| END | Endurance  | Résistance physique, capacité à encaisser |
| INT | Intelligence | Analyse, connaissances, raisonnement |
| SAG | Sagesse    | Perception, jugement, intuition |
| PRE | Précision  | Tir, visée, exactitude des armes à distance |
| TEC | Technique  | Maîtrise technique, armes flexibles, exécution propre |
| VOL | Volonté    | Résistance mentale, détermination |
| CHA | Charisme   | Influence sociale, présence |
| MAG | Magie      | Usage et défense magique |

## Jets d20 opposés

- Attaque : `1d20 + statistique + bonus équipement`
- Défense possible :
  - AGI (esquive)
  - END (encaisser)
  - FOR (blocage arme)
  - MAG (défense magique)

## Dégâts

Les dégâts proviennent uniquement de l'arme ou d'une capacité. Les statistiques n'ajoutent jamais de dégâts.
