# BLOC 5.1 — Statistiques, Jets et Dégâts

## Statistiques officielles

| Abréviation | Statistique  | Rôle principal                                                      |
|-------------|--------------|---------------------------------------------------------------------|
| FOR         | Force        | Puissance physique, blocage, armes lourdes ou longues.              |
| AGI         | Agilité      | Esquive, initiative, déplacement.                                   |
| END         | Endurance    | Résistance physique, capacité à encaisser.                          |
| INT         | Intelligence | Analyse, connaissances, raisonnement.                               |
| SAG         | Sagesse      | Perception, jugement, intuition.                                    |
| PRE         | Précision    | Tir, visée, exactitude des armes à distance.                        |
| TEC         | Technique    | Maîtrise technique, armes flexibles, exécution propre.              |
| VOL         | Volonté      | Résistance mentale, détermination.                                  |
| CHA         | Charisme     | Influence sociale, présence.                                        |
| MAG         | Magie        | Usage et défense magique.                                           |

## Jets d20 opposés

- Attaque : `1d20 + statistique appropriée + bonus d'équipement`.
- Défense possible :
  - AGI pour esquive
  - END pour encaisser
  - FOR pour blocage avec arme
  - MAG pour défense magique
- Le choix de la statistique dépend de la nature de l'action ou de la défense.

## Dégâts

- Les dégâts proviennent uniquement d'une arme ou d'une capacité.
- Les statistiques n'ajoutent jamais directement de dégâts.
- Les bonus d'équipement ou effets spéciaux peuvent modifier la résolution selon la règle concernée.
